A plugin to show virtual YouTube video on the wordress page by using shortcode [rcode_video_ads].

== Description ==
A plugin to show virtual YouTube video on the page and able to realize a video watching experience with several advertisements, with 'skip ad in 5 sec' button on the screen, just like YouTube. <br />
Please use shortcode <strong>[rcode_video_ads]</strong>

You can see demo here - <a href="http://www.rcodehub.com/?page_id=282" target="_BLANK">Click Here</a>

== Installation ==

1. Upload the `video-with-ads` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to the video-with-ads.
